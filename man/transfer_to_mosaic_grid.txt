transfer_to_mosaic_grid(1)
==========================

NAME
----

transfer_to_mosaic_grid - transfer old grid files to the current mosaic grid

SYNOPSIS
--------
[verse]
transfer_to_mosaic_grid --input_file _input_file_

DESCRIPTION
-----------

Transfers previous version grid file (The grid file has field x_T.y_T
) into a mosaic grid. The input file could be a ocean only grid or a
coupled grid.

OPTIONS
-------

*--input_file* _input_file_::

The file name of previous version grid.
